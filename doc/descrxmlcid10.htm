Descrição do arquivo CID10.XML

O arquivo CID10.XML contém, como visto, a estrutura, os códigos, as descrições e outros atributos dos capítulos, grupos, categorias e subcategorias da CID-10. Ele não contém, até o momento, notas, termos de inclusão, termos de exclusão e tabelas.

Sua estrutura é a seguinte:

  • O documento cid10 é composto de um conjunto de elementos capitulo.

  • Cada elemento capitulo é composto de:

    • atributo numcap: número do capítulo em arábico; se igual a 0, indica que o capítulo contém classificações extras;
    • atributo romano: número do capítulo em romano; vazio se numcap = 0;
    • atributo inicial: código (codcat) da primeira categoria do capítulo;
    • atributo final: código (codcat) da última categoria do capítulo; e
    • atributo codcap: código do capítulo, usado como identificador, montado como inicial-final;
    • elemento nome: texto com o nome do capítulo;
    • elemento nome50: texto com o nome abreviado do capítulo (50 caracteres); e
    • conjunto de um ou mais elementos grupo e/ou categoria.

  • Cada elemento grupo é composto de:

    • atributo inicial: código (codcat) da primeira categoria do grupo;
    • atributo final: código (codcat) da última categoria do grupo;
    • atributo codgrupo: código do grupo, usado como identificador, montado como inicial-final;
    • elemento nome: texto com o nome do grupo;
    • elemento nome50: texto com o nome abreviado do grupo (50 caracteres); e
    • conjunto de um ou mais elementos grupo e/ou categoria (portanto, um grupo pode conter outros grupos).

  • Cada elemento categoria é composto de:

    • atributo codcat: código da categoria, usada como identificador, no formato LNN, sendo L uma letra e NN um número de 01 a 99;
    • atributo ehsubcat: indica (sim ou nao) se a categoria é também uma subcategoria, ou seja, não tem subcategorias; o default é nao;
    • elemento opcional duplaclassificacao: indica se a categoria faz parte da classificação cruz ou asterisco; ver adiante;
    • elemento nome: texto com o nome da categoria;
    • elemento nome50: texto com o nome abreviado da categoria (50 caracteres);
    • elemento opcional restricoes: indica as restrições da categoria (só usado se ehsubcat=sim); ver adiante; e
    • conjunto de uma ou mais elementos subcategoria.

  • Cada elemento subcategoria é composto de:

    • atributo codsubcat: código da subcategoria, usada como identificador, no formato LNNS, correspondendo LNN ao código da categoria a que a subcategoria pertence e S à subdivisão da categoria;
    • elemento opcional duplaclassificacao: indica se a subcategoria faz parte da classificação cruz ou asterisco; ver adiante;
    • elemento nome: texto com o nome da subcategoria;
    • elemento nome50: texto com o nome abreviado da subcategoria (50 caracteres);
    • elemento opcional restricoes: indica as restrições da subcategoria; ver adiante;
    • conjunto opcional de elementos categoriasabsorvidas: indica que categorias/subcategorias são agora referenciados por esta subcategoria; ver adiante.

  • Cada elemento restricoes é composto de:

    • atributo causaobito: indica (sim ou nao) se a categoria/subcategoria pode causar óbito; o default é sim; e
    • atributo sexo: indica se a categoria/subcategoria está restrita a homens (apenas_homens), a mulheres (apenas_mulheres) ou se não há restrição (ambos); o default é ambos.

  • Cada elemento categoriasabsorvidas é composto de:

    • atributo codsubcat: código da categoria/subcategoria extinta agora codificada na categoria/subcategoria referenciada.

  • Cada elemento duplaclassificacao é composto de:

    • atributo tipo: indica o tipo de classificacao a que a categoria/subcategoria se refere: cruz ou asterisco; e
    • conjunto opcional de elementos referencia, com as categorias/subcategorias segundo a outra classificação.

  • Cada elemento referencia é composto de:

    • atributo codrefer: indica o código da categoria/subcategoria segundo a outra classificação;
    • texto descrevendo a referência, normalmente o próprio atributo codrefer, editado, seguido de uma cruz (†) ou um asterisco (*).
